# 山口の旅を選ぶ — スワイプ体験選びアプリ

豊かな旅 ＝ 帰った後の日常に残り続ける旅。  
山口県の体験50件をTinder風スワイプで選んで、あなただけの旅を設計するアプリです。

## 機能

- 🎴 **スワイプUI** — 右で「気になる」、左でスキップ
- 📍 **エリア・カテゴリフィルター** — 萩・長門・下関など10エリア、6カテゴリ
- ⭐ **豊かさスコア4軸** — 人との接点・余白・持ち帰り・五感
- 💭 **気になるリスト** — 選んだスポット一覧
- ✦ **旅タイプ診断** — 3件以上選ぶと旅人タイプが分かる

## ローカルで動かす

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開く。

## Vercelにデプロイする手順

1. このフォルダをGitHubにアップロードする
2. https://vercel.com にログイン（GitHubアカウントで可）
3. 「New Project」→ GitHubのリポジトリを選択
4. 設定はデフォルトのまま「Deploy」ボタンを押す
5. 数分後にURLが発行される → QRコードを作って公有

## 技術スタック

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- カスタムスワイプ実装（framer-motion不要の軽量版）

## スポットデータ

`src/data/spots.ts` に50件。追加は同じフォーマットで配列に追加するだけ。
